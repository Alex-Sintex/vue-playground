# Vue Playground

Collection of my Vue practice projects and tutorials.  
Each folder contains a small demo focused on a specific concept (e.g., reactivity, computed properties, lifecycle hooks, Vuex, localStorage).

## 📂 Structure

- `vue2-fruits-basics` → simple data binding and list rendering
- `vue2-background-reactivity` → dynamic class binding with reactivity
- `vue2-localstorage-todo` → todo list with localStorage persistence
- `vue2-computed-basics` → computed properties (string reverse, counter logic)
- `vue2-lifecycle-hooks` → lifecycle hooks demo
- `vue2-vuex-basics` → Vuex state management demo
- (more folders will be added over time)

## 🎯 Purpose

This repository is a sandbox for learning and experimenting with Vue and related full‑stack concepts.  
It is not intended as a polished product, but as a record of my progression and practice.

## 🛠️ Tech Stack

- Vue 3
- Vuex
- Vite (for some projects)
- Nuxt / Quasar (in later demos)
